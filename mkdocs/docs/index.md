# KoalaTs Docs Platform

This is the MkDocs/Material bootstrap for the KoalaTs documentation migration.

It exists alongside the current Docusaurus site while the migration is in progress.

The final target will provide:

- a shared landing page at `/`
- versioned docs at `/docs/2.x/` and `/docs/1.x/`
- manual GitHub Pages deployment from `main`
- a Dockerized local workflow

This bootstrap PR does not cut over production.
