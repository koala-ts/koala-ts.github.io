Welcome to KoalaTs. KoalaTs is built for backend developers who want a TypeScript framework with explicit HTTP primitives, predictable control flow, and minimal ceremony.

## Getting Started

Use the [KoalaTs CLI](https://www.npmjs.com/package/@koala-ts/cli) to generate a new project:

```bash
npx @koala-ts/cli create my-app
```

This command creates a new project in the `my-app` directory.

Then install dependencies and start the server:

```bash
cd my-app
npm install
npm run start
```
